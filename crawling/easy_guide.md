# 📘 FAQ 데이터 추출하기: 아주 쉬운 설명서

우리가 지금 하려는 일은 **"kia.json이라는 파일 안에 들어있는 질문과 답변들을 꺼내서, 데이터베이스(MySQL)라는 표에 예쁘게 정리해 넣는 것"**이에요.

> 💡 **비유하자면?**  
> 마치 서류 뭉치(JSON 파일)에서 필요한 내용만 쏙쏙 뽑아서 엑셀 표(MySQL)에 정리하는 것과 같아요.

---

## 🛠️ 사용한 도구들 (준비물)

이 작업을 하기 위해 몇 가지 도구(라이브러리)를 사용했어요.

1.  **📦 json**  
    서류 뭉치(JSON 파일)를 읽을 수 있게 해주는 도구예요.
2.  **🔌 pymysql**  
    파이썬이 데이터베이스(MySQL)와 대화할 수 있게 해주는 통역사 같은 도구예요.
3.  **🧹 BeautifulSoup (bs4)**  
    답변 내용 중에 `<p>`, `<b>` 같은 컴퓨터 언어(HTML 태그)가 섞여 있는데, 이것들을 싹 걷어내고 우리가 읽을 수 있는 글자만 남겨주는 **'청소부'** 역할을 해요.

---

## 📝 코드가 하는 일 (단계별 설명)

제가 만들어드린 `json_to_mysql.py` 파일은 다음과 같은 순서로 일을 해요.

### 1단계: 📂 파일 열기
먼저 `kia.json`이라는 서류 뭉치를 엽니다. 그리고 그 안에서 `faqList`라는 이름표가 붙은 곳을 찾아가요. 거기에 질문과 답변들이 모여 있거든요.

### 2단계: 🧹 하나씩 꺼내서 청소하기
질문과 답변 세트를 하나씩 꺼냅니다.
*   **질문**: 그대로 가져옵니다.
*   **답변**: 원래 답변에는 글자 색이나 크기를 꾸며주는 'HTML 태그'라는 게 붙어 있어서 지저분해 보여요. 그래서 **BeautifulSoup**이라는 청소부를 시켜서 깨끗한 글자만 남깁니다.

### 3단계: 💾 표에 적어 넣기 (DB 저장)
이제 깨끗해진 질문과 답변을 데이터베이스(MySQL)에 저장합니다.
*   `db_config`: 데이터베이스라는 창고에 들어가기 위한 열쇠 정보(아이디, 비밀번호 등)예요. **여기를 본인 정보로 꼭 채워주셔야 해요.**
*   `INSERT INTO...`: "이 질문과 답변을 표에 넣어줘!"라고 명령하는 부분이에요.

---

## 🚀 실행하는 방법

### 1️⃣ 설치하기
청소부(BeautifulSoup)와 통역사(pymysql)가 없다면 먼저 데려와야 해요. 터미널(검은 화면)에 아래 명령어를 입력하세요.

```bash
pip install pymysql beautifulsoup4
```

### 2️⃣ 정보 입력하기
`json_to_mysql.py` 파일을 열어서 `db_config`라고 써진 부분에 본인의 아이디와 비밀번호를 적어주세요.

### 3️⃣ 실행하기
이제 "시작!" 버튼을 누르는 것과 같아요.

```bash
python json_to_mysql.py
```

이렇게 하면 서류 뭉치에 있던 내용들이 자동으로 깔끔하게 정리되어 데이터베이스에 저장될 거예요! 🎉
